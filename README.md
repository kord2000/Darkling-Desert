# Darkling-Desert
A small top-down shooter game created using q5play and q5.js. 
Originally created using the [p5.js web editor](https://editor.p5js.org/DrewBalbuena/sketches/XP_bUg31u). I have migrated it to this repo to continue development. See my journey as I continue creating and learning. 

**What I have learned:** 
- Using objects to track program state.
- OOP practices.
- Closures in Javascript.
